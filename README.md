# Maxwell Construction
This is a python based code to obtain saturation pressure, liquid & vapor density for a given saturated pressure.

## Supported Equation of State:
- Carnahan-Starling (CS)
- Peng-Robinson (PR)
- Redlich-Kwong (RK)
- Redlich-Kwong-Soave (RKS)
- Van der Waals (VW)
